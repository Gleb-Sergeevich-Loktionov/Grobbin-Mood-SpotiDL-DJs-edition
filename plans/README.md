## Context

AI-scaffolded ("Made with Bob") Python CLI for downloading Spotify playlists via
YouTube. **It does not run from a clean checkout.** Multiple modules import a
`src.shared.lib` package that has never existed in git history, and
`metadata_handler.py` raises `NameError` at import time. The README describes
successful runs the code cannot currently produce.

These plans restore the app to a runnable, testable state and fix documented bugs.
